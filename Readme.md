# Bolt Dev — Creative Developer Portfolio

Welcome to the official repository for my portfolio. I am **Bolt Dev**, a 13-year-old developer passionate about creative coding and modern web technologies.

## About the Project

This portfolio was built to showcase my projects and technical skills. It combines high-end UI/UX design with interactive elements.

- **Frontend:** HTML5, CSS3 (Custom Properties, Grid & Flexbox)
- **Animations:** GSAP (GreenSock Animation Platform)
- **3D Elements:** Three.js / WebGL
- **Tools:** Figma for design, Node.js for infrastructure

## Expertise

- **Creative Development:** Immersive browser experiences using WebGL & Shaders.
- **Backend:** Desktop apps with Wails (Go) and Discord bots with Pycord.
- **Modern Web:** React & Next.js for scalable applications.
